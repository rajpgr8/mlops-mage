# Setup

1. Clone repo:

   ```
   git clone https://github.com/mage-ai/mlops.git
   ```

1. Change directory into the cloned repo:

   ```
   cd mlops
   ```

1. Launch Mage and the database service (PostgreSQL):

   ```
   ./scripts/start.sh
   ```

1. Open [`http://localhost:6789`](http://localhost:6789) in your browser.
